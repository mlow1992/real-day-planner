# Work Day Scheduler

## Purpose
To be able to keep track of daily tasks as a person goes about their workday. Each task entered should remain consistent until updated or cleared from local storage.

## Built With
*HTML
*CSS
*Bootstrap
*JavaScript
*Jquery
*Moment.js
